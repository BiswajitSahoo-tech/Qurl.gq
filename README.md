# Setup

Clone the repo
```bash
git clone https://github.com/TheRedstoneRadiant/Qurl.gq
cd Qurl.gq
```

Copy .env.example and replace variables
```bash
cp .env.example .env
nano .env
```

Install dependencies
```bash
npm install
```

Start development server
```bash
npm run dev
````
